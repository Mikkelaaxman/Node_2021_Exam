Variables are like little data containers, we can declare a variable in javascript by writing:
let myVar = 1

We can define loops in javascript by writing:
const failures = ["misclick", "falling", "tiktok fail"];

and then iterate over the loop like this:

failures.map((fail, index) => { console.log(fail, index);
});

We can define a object in javascript like this:
const myObject = { attribute: "value" };

We can define a function in javascript like this:
const sayHi = () => { console.log("hi") };
