The basics Node & NPM Express Environment

enviroment variables are, just like the name suggest, variables that are global to the your enviroment. as an example if you want to look at the enviorment variable for your port you can simply write process.env.port that will output the environt variable PORT


another thing that has something to do with enviroment is the package cross-env cross-env allows us to run our scripts from package.json in all platforms simply write cross-env before your script and make sure you have the package installed
