The basics Node & NPM Express Environment

Node.js is a back-end javascript runtime enviroment that allows us to execute javascript code outside the browser to check if you have node correctly installed simply open a terminal and type node -v if it outputs a number you done it right


npm also known as node package manager, is a package manager for javascript npm is also the default package manager for node.js you can check if you have node installed simply open a terminal and type npm -v if it outputs a number you done it right npm consist of a client, and a large database of packages you can create a package.json by running the init command: npm init
