The basics Node & NPM Express Environment

In this course, we have ( so far ) been focusin on the minimal web-framework Express. This framework helps us build and serve rest api's, among other things. We can install Express by navigating to our package.json and then in the terminal writing npm install express and then installing with npm i In out app.js file we can import Express with the following line of code const express = require("express"); and then instanciate it with const app = express(); now that we have express in our app.js lets use it, a simple way of doing this is just by listen on port 8080, we can do this like so app.listen(8080); But when we go to localhost:8080 it dosent show us anything usefull, lets fix that with the following lines of code
app.get("/", (req, res) => {
res.send({});
});
